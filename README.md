# 📖 About

Raj Tuition Classes (RTC) is a complete Tuition Management System designed to digitize coaching institute operations.

The application replaces traditional registers and spreadsheets with a modern web application where teachers manage students, attendance, fees, tests, notices, study materials and classes while students can track their complete academic information from a dedicated dashboard.

The project is currently running in production and is being used by real teachers and students.

# 🔥 Challenges Faced

This project was not built in one go.

Some of the major engineering challenges included:

* Designing a scalable PostgreSQL schema
* Building attendance logic without duplicate records
* Monthly fee generation
* React Query caching
* PWA optimization
* Xiaomi device rendering issues
* HEIC image upload support
* Supabase RLS understanding
* Production deployment
* Mobile responsiveness
* Real-world bug fixing

Every issue was solved through iterative development, testing and feedback from actual teachers.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/rtc.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production

```bash
npm run build
```
